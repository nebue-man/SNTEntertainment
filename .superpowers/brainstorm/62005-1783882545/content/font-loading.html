How should we handle the Lausanne typeface?

Lausanne is a commercial font from Swiss Typefaces — the design.md specifies it as the only typeface used across the entire site.

A

System-ui fallback for now (Recommended)

Wire up the CSS custom property and Tailwind @theme block exactly as specified in design.md, but use the ui-sans-serif, system-ui fallback stack until you provide the font files. Swapping in the real Lausanne later is a one-line change to globals.css. Keeps development unblocked immediately.

B

Self-hosted from the start

You provide the .woff2 files and we load them via @font-face in globals.css. The site looks exactly correct immediately — correct kerning, weights, and rhythm from day one. Requires you to have the license and files ready now.
